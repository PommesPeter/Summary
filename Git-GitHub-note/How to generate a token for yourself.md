# 生成Github Token

## 第一步：进入官网
点击头像旁边的小箭头--->选择“Settings”
![在这里插入图片描述](https://img-blog.csdnimg.cn/20200317094003465.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3dlaXhpbl80NTcwOTMzMA==,size_16,color_FFFFFF,t_70)

## 第二步：进入Settings之后
点击左侧菜单下的“Developer settings”
![在这里插入图片描述](https://img-blog.csdnimg.cn/20200317094125554.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3dlaXhpbl80NTcwOTMzMA==,size_16,color_FFFFFF,t_70)
## 第三步
点击选择“Personal access tokens”:
![在这里插入图片描述](https://img-blog.csdnimg.cn/20200317094302434.jpg)
## 第四步
点击“Generate a personal access token”:
![在这里插入图片描述](https://img-blog.csdnimg.cn/20200317094340132.jpg)
## 第五步
把以下选项全部勾选，然后点击下方绿色按钮“”General token，便可生成一个token。
![在这里插入图片描述](https://img-blog.csdnimg.cn/20200317094416117.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3dlaXhpbl80NTcwOTMzMA==,size_16,color_FFFFFF,t_70)
